# BrighterTools.Payments

Shared payment abstractions and provider implementations for BrighterTools applications.

## Packages
- `BrighterTools.Payments.Stripe`: Stripe Checkout, Connect, balance, settlement, and webhook signature helpers.


## Development
- Build: dotnet build BrighterTools.Payments.slnx -c Release`r
- Test: dotnet test BrighterTools.Payments.slnx -c Release`r
